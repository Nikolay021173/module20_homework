# Домашние задания модуля 20 "Стандарты написания кода и общие подходы".

В папке **KISS-and-DRY**, рассмотренны 2 подхода к улучшению написания кода на **JS KISS и DRY**. В файле **original-version.js** содержится код написанный неправильно с точки зрения принципов **KISS и DRY**. В файле **improved-version.js** содержится код исправленный согласно принципам **KISS и DRY**, все пояснения находятся в файлах папки **KISS-and-DRY** в **закомментированном виде**. 

В папки **YAGNI** в файле **yagni-principle.js** в закомментированном виде наглядно показанно нарушение принципа **YAGNI** (**лишний функционал** (см. закомментированную часть кода в файле **yagni-principle.js**)).

В папке **БЭМ** в файле **index.html** показаны правильный и неправильный (в закомментированном виде с пояснением) подходы в верстке фрагмента сайта согласно принципу **БЭМ**.

В папке **SOLID** в файле **solid-principle.ts**, в коде написанном на **TypeScript**, показанно применение данного принцип на примере реализации интерфейсов и классов для 2-х мерных геометрических фигур (все пояснения находятся в закомментированном виде в файле **solid-principle.ts**)

В папке **Семантика-и-Валидность**, в файле **index.html** на примере вёрстки фрагмента сайта наглядно показанны **правильный и неправильный (закоммендированный код вёрстки) подходы** с точки зрения семантической вёрстки, все пояснения в комментариях в файле **index.html**. **Валидность** вёрстки была проверенна с помощью **онлайн-сервиса W3C** (пояснения в комментариях файла **index.html**).
